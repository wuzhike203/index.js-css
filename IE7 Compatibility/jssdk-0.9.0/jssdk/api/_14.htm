Library: Set.js
Overview
jsx.collections.Set is an Interface class. Defines the methods for base set. A set is a collection that contains no duplicate elements.
  • source: Set.js
  • example: 
    		class MySet implements Set
Constructors
jsx.collections.Set()
methods
  • add(_object) - Adds the specified element to this set if it is not already present. Returns true if the element was not present before adding.
  • addAll(_collection) - Adds all of the elements in the specified collection to this set if they're not already present. Returns whether the this set was modified.
  • clear() - Removes all elements in the collection.
  • contains(_object) - Returns whether the specified object is contained in the collection.
  • containsAll() - Returns whether the elements in the specified collection exist in the collection.
  • isEmpty() - Returns whether the collection is empty.
  • remove(_object) - Removes the specified object from the collection. Return whether the object was removed.
  • removeAll(_collection) - Removes all the elements in the specified collection from the current collection. Returns whether the current collection was modified.
  • retainAll(_collection) - Removes all elements in the current collection if they are not in the specified collection. Returns whether the current collection was modified.
  • size() - Returns the size of the collection.
