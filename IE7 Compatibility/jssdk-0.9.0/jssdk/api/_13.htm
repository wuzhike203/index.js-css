Library: Map.js
Overview
jsx.collections.Map is an Interface class. Defines the methods for base map. A map is an object that maps keys to values.
  • source: Map.js
  • example: 
    		class MyMap implements Map
Constructors
jsx.collections.Map()
methods
  • clear() - Clears all the keys and values in the map
  • containsKey(_key) - Returns whether the map contains the specified key.
  • containsValue(_value) - Returns whether the map contains the specified value.
  • entrySet() - Returns a set view of the mappings contained in this map.
  • get(_key) - Returns the value to which this map maps the specified key.
  • isEmpty() - Returns whether if this map contains no key-value mappings.
  • keySet() - Returns a set view of the keys contained in this map.
  • put(_key, _value) - Associates the specified value with the specified key in this map. Returns previous value associated with specified key, or null if there was no mapping for key.
  • putAll(_map) - Copies all of the mappings from the specified map to this map.
  • remove(_key) - Removes and returns the mapping for this key from this map if it is present.
  • size() - Returns the number of key-value mappings in this map.
  • values() - Returns a collection view of the values contained in this map.
