methods
  • addAt(_index, _object) - Add an element to the list at a specified index. Returns whether the element was added into the list.
  • addAllAt(_index, _collection) - Add all the elements in the specified collection starting at the specified index. Returns whether the current list was modified.
  • get(_index) - Returns an element from the list at the specifed index.
  • indexOf(_object) - Returns the index of the first occurance of the specified object. If the specified object does not exist in the collection, -1 will be returned.
  • lastIndexOf(_object) - Returns the index of the last occurance of the specified object. If the specified object does not exist in the collection, -1 will be returned.
  • removeAt(_index) - Removes and returns the element from the collection at the specifed index.
  • set(_index, _object) - Assigns the specified object to the collection at the specified index. If an object is already living at the index, it is replaced.
  • subList(_fromIndex, _toIndex) - Returns a view of the portion of this list between the specified fromIndex, inclusive, and toIndex, exclusive.
