Constructors
jsx.lang.Thread(_block, _object, _args, _milliseconds, _name)
This is not a true thread. This is just a cleaner API wrapping the global objects setTimeout and setInterval methods.
parameters
Function _block Block to execute.
Object _object Context object.
Array _args Array of parameters to pass into the block.
Number _milliseconds Specify in milliseconds when to execute. If the thread is a daemon thread, the milliseconds will indicate the intervals the block will execute. If the thread is not a daemon thread, the milliseconds will indicate the delay until the block will execute.
String _name Name of the Thread.
methods
  • start() - Starts the jsx.lang.Thread.
  • stop() - Stops the jsx.lang.Thread.
  • sleep(_milliseconds) - Makes the Thread sleep for a number of milliseconds.
  • getID() - Returns the Thread ID.
